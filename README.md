# EduCaitors - AI-Assisted Grading Platform

An intelligent grading platform for educators that combines AI-powered evaluation with human oversight.

## Features

- 🤖 **AI Evaluation Engine** - Automated grading with confidence scoring
- 📱 **Mobile Responsive** - Grade on-the-go with full mobile support
- 📝 **Bulk Grading** - Process assignments efficiently with batch operations
- ⚖️ **Appeals System** - Handle student disputes with AI-assisted triage
- 📊 **Analytics Dashboard** - Track grading progress and performance metrics

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Styling**: Tailwind CSS v4 + HeroUI v3
- **State Management**: Zustand
- **Animation**: Framer Motion
- **Icons**: Gravity UI

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is configured for deployment on **Vercel** with GitHub Actions CI/CD.
